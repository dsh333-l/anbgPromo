# 目标用户画像 (audience.md)

> 版本：v1.0 | 用途：trend-hunter 判断选题相关性、copywriter 选择场景切入角度。

---

## 核心人群优先级

按转化可能性从高到低排列：

---

### Tier 1：新装修年轻家庭（最高优先）

- **典型画像**：25-40岁，正在装修或3个月内准备装修
- **消费能力**：愿意为居住质量加预算，但对"花冤枉钱"敏感
- **关注点**：空气质量、通风、噪音、外观、性价比
- **内容触点**：搜"装修避坑""窗户怎么选""系统窗值不值"
- **情绪特征**：信息焦虑，怕选错但又不懂怎么选
- **转化路径**：内容种草 → 评论区互动 → 私信留户型 → 到店

### Tier 2：有娃家庭

- **典型画像**：家有0-10岁儿童，对安全和健康极度敏感
- **关注点**：儿童安全锁、防坠、抗菌、通风换气
- **内容触点**：搜"儿童安全窗""高层有小孩怎么选窗"
- **情绪特征**：宁可多花钱也不能出事，但需要被告知"风险在哪"
- **转化路径**：安全类内容建立信任 → 咨询具体配置 → 到店体验

### Tier 3：高层 / 沿海 / 台风区家庭

- **典型画像**：住15楼以上，或沿海城市、台风多发区
- **关注点**：抗风压、防坠落、密封性、型材强度
- **内容触点**：搜"高层窗户安全""台风天窗户响"
- **情绪特征**：经历过大风天窗户异响后开始重视
- **转化路径**：痛点内容命中 → 自查现有窗户 → 换窗咨询

### Tier 4：临街/噪音困扰住户

- **典型画像**：临马路、临高架、靠学校/工地
- **关注点**：隔音、睡眠质量、密封系统
- **内容触点**：搜"隔音窗""睡眠差是不是窗户问题"
- **情绪特征**：已经被噪音折磨，急需解决方案
- **转化路径**：隔音内容 → 测试方法教育 → 换窗咨询

### Tier 5：改善型 / 高审美用户

- **典型画像**：二次装修或对居住品质有更高追求
- **关注点**：极窄边框、大玻璃、采光、整体美学
- **内容触点**：搜"极简风窗户""窄边框系统窗"
- **情绪特征**：知道好的东西贵，但希望"贵得有道理"
- **转化路径**：审美类内容种草 → 案例对比 → 到店看实物

### Tier 6：智能家居爱好者

- **典型画像**：已有或正在组建智能家居系统
- **关注点**：远程控窗、联动场景、自动化
- **内容触点**：搜"智能窗""窗户联动小米/华为"
- **情绪特征**：对新技术有热情，愿意为"酷"买单
- **转化路径**：智能场景内容 → 了解联动方案 → 定制咨询

## 写作时的人群适配规则

| 内容支柱 | 优先面向人群 | 开头锚定方式 |
|---------|------------|-------------|
| 健康人居 | Tier 1 + Tier 2 | "如果你家刚装修/家有小孩" |
| 高层安全 | Tier 2 + Tier 3 | "住在X楼以上的" |
| 静音节能 | Tier 4 + Tier 1 | "如果你家临街/临高架" |
| 智能窗 | Tier 6 + Tier 1 | "家里已经装了智能家居的" |
| 审美升级 | Tier 5 + Tier 1 | "如果你在意采光和视觉整洁度" |
