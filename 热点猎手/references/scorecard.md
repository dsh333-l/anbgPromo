# 选题评分卡 (scorecard.md)

> 版本：v1.0 | 用途：热点猎手对每个候选选题打分，决定做/不做/备选。

---

## 评分维度（满分25分，每项1-5分）

### 1. 目标用户相关性（1-5分）

这个选题跟 audience.md 中的 Tier 1-3 人群有多强的关联？

| 分数 | 标准 |
|------|------|
| 5 | 直接命中 Tier 1 核心痛点（正在装修的年轻家庭） |
| 4 | 命中 Tier 2-3 明确痛点 |
| 3 | 命中 Tier 4-6 或泛家居人群 |
| 2 | 跟居住相关但跟门窗关联较弱 |
| 1 | 跟安贝格目标用户几乎无关 |

### 2. 传播性 / 反常识感（1-5分）

用户看到标题后，是否有"点进去看看"的冲动？

| 分数 | 标准 |
|------|------|
| 5 | 强反常识（"装修花40万，最后输在窗上"） |
| 4 | 有信息差或少有人讲的角度 |
| 3 | 常规但实用的科普 |
| 2 | 老生常谈，但换了个说法 |
| 1 | 无聊/无感/谁都说过 |

### 3. 可视化程度（1-5分）

这个选题能不能做成有画面感的短视频？

| 分数 | 标准 |
|------|------|
| 5 | 有强对比画面（如台风天窗户摇晃 vs 稳固） |
| 4 | 可以展示操作或测试过程 |
| 3 | 可以用动画/图文辅助 |
| 2 | 只能纯口播 |
| 1 | 很难做成视频 |

### 4. 转化潜力（1-5分）

看完这条内容的人，有多大概率产生咨询行为？

| 分数 | 标准 |
|------|------|
| 5 | 内容直接关联选购决策（如"怎么判断窗户该不该换"） |
| 4 | 能自然引导到产品咨询 |
| 3 | 建立信任但不直接产生咨询 |
| 2 | 纯科普，转化路径较远 |
| 1 | 看完没有任何行动冲动 |

### 5. 风险可控性（1-5分）

这个选题有没有踩红线的可能？

| 分数 | 标准 |
|------|------|
| 5 | 完全无风险，纯科普/生活场景 |
| 4 | 涉及产品对比但不点名，风险低 |
| 3 | 涉及安全话题但可以控制措辞 |
| 2 | 涉及价格、竞品或敏感事件，需谨慎 |
| 1 | 高概率踩到 taboo.md 红线 |

---

## 判定标准

| 总分 | 决策 | 说明 |
|------|------|------|
| 20-25 | **优先做** | 今天第一批排产 |
| 16-19 | **可做** | 列入备选，产能有余时做 |
| 12-15 | **暂存** | 放入常青选题库，等合适时机 |
| 11以下 | **放弃** | 不做 |

## 特殊规则

- 任何单项得1分的选题，即使总分达标也需人工复核
- "风险可控性"得1分或2分的选题，即使总分高也必须人工确认
- 连续3天没有20分以上选题时，从 evergreen-topics.md 中抽取

## 评分输出模板

```
选题：[标题]
来源：[热点/常青/用户提问]
日期：YYYY-MM-DD

1. 用户相关性：X/5 — [一句话理由]
2. 传播性：X/5 — [一句话理由]
3. 可视化：X/5 — [一句话理由]
4. 转化潜力：X/5 — [一句话理由]
5. 风险可控：X/5 — [一句话理由]

总分：XX/25
判定：优先做 / 可做 / 暂存 / 放弃
推荐支柱：[健康人居/高层安全/静音节能/智能窗/审美升级]
推荐形式：[15秒短口播/90秒案例拆解/30秒情绪钩子/图文长帖]
15秒切口：[如果先做15秒版，最应该先讲的那个点]
```
