# 安贝格禁用表达清单 (taboo.md)

> 版本：v1.0 | 用途：所有 Agent 生成或审核内容时必须检查。
> 命中"红线"即退回重写；命中"黄线"需人工确认。

---

## 一、红线：绝对禁用（出现即退回）

### 1.1 虚假绝对化

- 全网第一 / 行业第一 / 行业领先 / 遥遥领先
- 绝对零甲醛 / 百分百隔音 / 完全隔热 / 永不漏水 / 永不变形
- 最安全 / 最安静 / 最好 / 最强 / 最高端（无第三方数据时）
- 碾压所有品牌 / 秒杀同行 / 吊打XX
- 国家级 / 世界级 / 军工级 / 航天级（无真实认证时）
- 无毒无害 / 绝对环保 / 零污染

### 1.2 促销逼单

- 最后X个名额 / 仅剩X套
- 今天不买明天涨价 / 过了这村没这店 / 错过等一年
- 限时特惠 / 秒杀价 / 跳楼价 / 工厂直供价
- 现在下单立减 / 交定金抵XX
- 赶紧私信 / 赶紧抢 / 手慢无

### 1.3 恐吓话术

- 不装这个窗就是拿家人的命赌博
- 你家孩子随时可能出事
- 每天都在吸毒气你知道吗
- 等出了事就晚了 / 不换窗全家完蛋

### 1.4 攻击竞品

- 点名攻击任何品牌（如"XX品牌就是垃圾"）
- 暗示性贬低（如"不像某些品牌偷工减料"）
- 未经核实的竞品负面信息

### 1.5 网红腔 / 低质表达

- 家人们 / 宝子们 / 姐妹们
- 绝绝子 / yyds / 无语子 / 真的会谢
- 哈哈哈哈 / 懂的都懂

### 1.6 官腔八股

- 本公司致力于 / 我司产品 / 尊敬的消费者
- 赋能美好生活 / 引领行业发展
- 全方位 / 多维度 / 深层次（空泛修饰堆砌）

## 二、黄线：谨慎使用（需人工确认）

### 2.1 "最"字

- 有第三方检测数据支撑：✅ 可用，需标注来源
- 品牌内部对比：✅ "安贝格目前隔音最好的一款"
- 无数据的泛化使用：❌

### 2.2 数据引用

- 必须标明来源（检测机构名称或报告编号）
- 实验室数据不得等同于实际效果
- ✅ "实验室条件下隔音量可达40dB"
- ❌ "安装后保证降噪40分贝"

### 2.3 健康声明

- 门窗不是医疗器械，不得暗示治疗效果
- ❌ "防过敏""治鼻炎""改善呼吸系统疾病"
- ✅ "带过滤的通风系统可以减少室内灰尘和PM2.5"

### 2.4 智能功能

- 不得暗示完全自主决策：❌ "窗户自己判断"
- 需说明触发条件：✅ "风雨传感器检测到下雨时自动关窗"
- 不得承诺100%可靠：❌ "永远不会忘关窗"

### 2.5 价格相关

- 不在公开内容中标注具体价格
- 不做"XX元/平方"的笼统报价
- ✅ "系统窗的价格差异主要来自型材、玻璃和五金"

## 三、敏感话题禁区

### 不评论的话题

房价涨跌、具体楼盘好坏、装修公司推荐、政治宗教争议。

### 安全话题规范

- 不得用事故新闻做恐吓素材
- 不放事故现场图片
- 可以讲"为什么安全配置重要"，语气平实
- ✅ "高层住户选窗时，防坠落配置需优先考虑"
- ❌ "去年XX小区又有孩子从窗户掉下去了！"

### 平台合规

- 抖音/视频号：不在口播中出现微信号、手机号
- 小红书：不得全是产品图，需包含场景图
- 引导私域用"评论区留言"而非"加微信"
