# 内容形式标准 (formats.md)

> 版本：v1.0 | 用途：文案写手根据选题特性选择合适的内容形式。

---

## 形式一览

| 编号 | 形式 | 时长/篇幅 | 适用场景 | 周频建议 |
|------|------|----------|---------|---------|
| F1 | 15秒短口播 | 12-18秒 | 单一观点、快速种草、短信息差 | 4-5条/周 |
| F2 | 90秒案例拆解 | 80-100秒 | 真实户型、前后对比 | 1-2条/周 |
| F3 | 30秒情绪钩子 | 25-35秒 | 强共鸣、强传播的短内容 | 1-2条/周 |
| F4 | 图文长帖 | 800-1500字 | 小红书/公众号深度内容 | 1条/周 |
| F5 | 评论区引导帖 | 50-100字 | 配合视频做私域转化 | 每条视频配1条 |

---

## F1：15秒短口播（默认形式）

### 结构模板

```
[0-3秒] 钩子：反常识 / 身份锚定 / 痛点直击
[3-7秒] 场景代入：用户遇到了什么
[7-11秒] 判断 / 原因：为什么会这样
[11-15秒] 结论 / 选择标准 / 互动收尾
```

### 适用支柱
全部五个支柱均适用

### 制作要求
- 字幕必须全程覆盖（很多用户静音刷视频）
- 一条只讲一个点，不展开第二层知识树
- 画面不能只有人脸，需穿插产品/场景 B-roll
- 语速控制在每秒 3-4 个字
- 用户未指定时长时，默认用这一形式
- 写作时要默认这条内容后续会被改写成 1 条即梦整段视频 prompt

---

## F2：90秒案例拆解

### 结构模板

```
[0-3秒] 钩子："这个家，窗选错了 / 选对了"
[3-20秒] 背景介绍：什么户型、什么楼层、什么问题
[20-50秒] 问题分析：原来的窗差在哪 / 新窗好在哪
[50-75秒] 前后对比：具体的改变（数据 + 感受）
[75-90秒] 总结 + 互动："你家是类似情况吗？"
```

### 适用支柱
P2（高层安全）、P3（静音节能）、P5（审美升级）

### 制作要求
- 最好有真实户型照片或视频素材
- 如果没有真实素材，可以用3D效果图 + 文字叙述
- 前后对比必须具体，不能只说"效果好了很多"

---

## F3：30秒情绪钩子

### 结构模板

```
[0-3秒] 强情绪开头：一句话戳中用户
[3-20秒] 快速展开：2-3个短句加强共鸣
[20-30秒] 留悬念 / 引导看完整版："详细拆解看上一条"
```

### 适用支柱
P1（健康人居）、P3（静音节能）最适合

### 制作要求
- 这是专门为传播设计的，不承担转化任务
- 目标是高播放、高完播率
- 可以做成系列："你以为___，其实___"
- 必须配合一条完整版内容一起发

---

## F4：图文长帖（小红书/公众号）

### 结构模板

```
[标题] 反常识 / 清单型 / 避坑型
[首图] 关键信息可视化（不能只放产品图）
[正文] 问题 → 原因 → 方案 → 推荐标准
[结尾] 互动引导 + 收藏引导
```

### 适用支柱
全部，但 P3（静音节能）和 P1（健康人居）在小红书上表现最好

### 制作要求
- 图片数量6-9张
- 第一张图必须有标题文字
- 正文分段，每段不超过3行
- 关键信息用加粗或色块标注

---

## F5：评论区引导帖

### 结构

每条视频发布后，用品牌号在评论区发一条引导帖：

```
模板A（咨询引导）：
"你家是高层、临街还是有娃家庭？不同情况选窗的优先级完全不同，评论你的情况帮你判断。"

模板B（资料引导）：
"评论'选窗'，整理了一份不同楼层的窗户配置建议，私信发你。"

模板C（讨论引导）：
"你家装修时在窗户上花了总预算的百分之几？评论聊聊。"
```

### 制作要求
- 每条视频必须配1条评论区引导
- 引导方向要跟视频内容匹配
- 不得在评论中放微信号或链接

---

## 形式选择规则

| 选题特征 | 推荐形式 |
|---------|---------|
| 有一个核心观点要传递 | F1（15秒口播，默认） |
| 有真实案例或前后对比素材 | F2（90秒案例拆解） |
| 话题自带情绪共鸣、适合传播 | F3（30秒情绪钩子）|
| 信息量大、需要图表或清单 | F4（图文长帖） |
| 需要引导到私域 | F5（评论区帖）配合以上任一形式 |
