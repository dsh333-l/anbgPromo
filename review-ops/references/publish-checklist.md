# 发布前检查清单 (publish-checklist.md)

> 版本：v1.0 | 用途：发布前最后一道关，review-ops 和你各过一遍。

---

## 发布包完整性检查

确认 `output/publish-packs/YYYY-MM-DD-topic/` 下包含：

- [ ] 01-idea.md — 选题评估
- [ ] 02-script.md — 终版脚本
- [ ] 03-storyboard.md — 分镜脚本
- [ ] 04-captions.srt — 字幕文件
- [ ] 05-cover-prompt.md — 封面描述/封面图
- [ ] 06-video-url.txt — 视频文件路径或链接
- [ ] 07-review.md — 审核报告（结果为✅通过）
- [ ] 08-publish-copy.md — 发布文案（标题+描述+标签+评论区）

---

| # | 检查项 | ✓ |
|---|--------|---|
| 1 | 视频画面无明显瑕疵（黑屏、卡顿、画面错位） | |

| 7 | 全片无 `references/taboo.md` 红线表达 | |
| 8 | 品牌提及方式自然（如有） | |

## 平台适配检查

### 抖音
- [ ] 视频比例 9:16 竖屏
- [ ] 无其他平台水印
- [ ] 话题标签使用 # 格式

### 视频号
- [ ] 视频比例 9:16 或 16:9
- [ ] 描述中无诱导关注表达

### 小红书（图文时）
- [ ] 图片6-9张
- [ ] 第一张图有标题文字
- [ ] 正文分段，每段≤3行

---

## 发布后动作

- [ ] 发布后10分钟内，用品牌号发评论区引导帖
- [ ] 在 data/content-log.md 记录发布时间和平台
- [ ] 在 data/performance-log.csv 中创建当天的行（数据待21:00回填）
