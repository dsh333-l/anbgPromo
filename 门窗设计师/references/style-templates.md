## 一、通用风格类型（可直接使用）

### S01 极简极窄边框

- **关键特征**：超窄边框（≤ 30mm）、隐藏式排水、框扇齐平
- **分格逻辑**：大面积玻璃，最少分格，追求通透
- **常配开启**：W01 固定 + W04 内开内倒
- **适合空间**：现代住宅、轻奢风、中古风
- **prompt 关键词**：极窄边框 / 框扇齐平 / 大面积通透玻璃 / 隐藏排水

### S02 现代细分格

- **关键特征**：标准窄边框（40-50mm），中等密度分格
- **分格逻辑**：3 联或 4 联，固定+开启交替
- **常配开启**：W01 固定 + W04 内开内倒
- **适合空间**：现代风、北欧风
- **prompt 关键词**：细分格 / 横向三联 / 干净线条

### S03 中式万字格

- **关键特征**：内嵌中式格栅或万字纹，外层为现代框
- **分格逻辑**：井字格、米字格、回字纹
- **常配开启**：W01 固定 + W04 内开内倒
- **适合空间**：新中式、简中风
- **prompt 关键词**：万字格 / 井字格 / 中式分格 / 内嵌格栅

### S04 法式拱顶分格

- **关键特征**：上方半圆或拱形，下方矩形分格
- **分格逻辑**：上拱下方，竖向 2-3 联
- **常配开启**：W01 固定 + W02/W03 平开
- **适合空间**：法式风、欧式风、别墅
- **prompt 关键词**：法式拱形 / 半圆顶 / 竖向窄分格

### S05 工业风窄边

- **关键特征**：黑色窄边框，密集横竖分格
- **分格逻辑**：横向 3-5 联 + 竖向 2-3 联
- **常配开启**：W01 固定为主 + 局部 W04
- **适合空间**：工业风、Loft、中古风
- **prompt 关键词**：黑框工业窗 / 密集分格 / 钢框感 / 横竖网格

### S06 日式简素

- **关键特征**：自然原木色或浅灰，横向分格
- **分格逻辑**：横向 2-3 联，比例和谐
- **常配开启**：W04 内开内倒 + W05 上悬
- **适合空间**：侘寂风、日式、原木风
- **prompt 关键词**：原木边框 / 横向分格 / 自然光感 / 素净

### S07 落地大单玻

- **关键特征**：极简边框，落地一整块玻璃
- **分格逻辑**：1 联落地或 2 联对开
- **常配开启**：D05 提升推拉门 / W01 全固定落地窗
- **适合空间**：现代风、轻奢风、观景房
- **prompt 关键词**：落地玻璃 / 极简边框 / 室内外通透

### S08 折叠开放式

- **关键特征**：多扇折叠收起，开放感最强
- **分格逻辑**：4-6 扇折叠
- **常配开启**：D06 折叠门
- **适合空间**：阳台、厨房吧台、半室外空间
- **prompt 关键词**：折叠门 / 全开放 / 多扇折叠收起

## 风格选择参考问题

如果客户犹豫，按这个顺序问：

1. 你家整体装修风格是？（简中/现代/轻奢/侘寂/法式/中古/北欧）
2. 喜欢通透大玻璃，还是有分格层次感？
3. 边框颜色想要黑、灰、白、还是木纹？
4. 有没有看到喜欢的图片可以发给我做参考？

把客户的回答映射到 S01-S08，给出 1-2 个推荐选项让客户确认。
