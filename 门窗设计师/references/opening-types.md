# 门窗开启方式选项 (opening-types.md)

> 版本：v1.0 | 用途：门窗设计师 Step 3 选择开启方式时调用。
> 说明：分"窗"和"门"两类，每类按使用场景做了适用建议。

---

## 一、窗类开启方式

| 编号 | 名称 | 简述 | 适用场景 |
|------|------|------|---------|
| W01 | 固定窗 | 不可开启，纯采光 | 高层观景位、采光辅助、楼梯间 |
| W02 | 平开窗（外开） | 整扇向外推开 | 阳台外侧、需要大通风量 |
| W03 | 平开窗（内开） | 整扇向室内开 | 不允许外开的高层、外立面要求干净 |
| W04 | 内开内倒 | 既能内开通风，又能上悬通风 | 主流系统窗，兼顾安全和通风 |
| W05 | 上悬窗 | 上方为轴小角度内倒 | 卫生间、厨房、雨天通风 |
| W06 | 下悬窗 | 下方为轴小角度外开 | 老式办公楼、地下室 |
| W07 | 推拉窗（左右） | 横向滑动开启 | 阳台封窗、传统装修 |
| W08 | 提升推拉窗 | 推拉前先抬升再滑动 | 大尺寸阳台门、密封性要求高 |
| W09 | 折叠窗 | 多扇折叠收起 | 厨房吧台、阳台开放式连通 |
| W10 | 中悬窗 | 中部为轴翻转 | 工业建筑、特殊采光位 |

## 二、门类开启方式

| 编号 | 名称 | 简述 | 适用场景 |
|------|------|------|---------|
| D01 | 平开门（单开） | 整扇侧轴开启 | 入户、卧室、书房 |
| D02 | 平开门（双开/对开） | 两扇对开 | 客厅阳台、宽洞口 |
| D03 | 推拉门（单轨） | 一扇滑动 | 厨房、衣帽间 |
| D04 | 推拉门（双轨/三轨） | 多扇滑动 | 客厅阳台、大开口分隔 |
| D05 | 提升推拉门 | 提升后滑动，密封性最好 | 高端阳台、节能要求高 |
| D06 | 折叠门 | 多扇折叠收起 | 阳台全开放、厨房半开放 |
| D07 | 极简地弹门 | 无明显边框，地面合页 | 玄关玻璃门、商业空间 |
| D08 | 旋转门 | 中轴旋转 | 别墅入户、特殊设计 |

## 常见组合（用于 Step 3 的"分格初步建议"）

| 洞口类型 | 推荐组合 |
|---------|---------|
| 客厅阳台门（W ≥ 2400mm） | D04 推拉 / D05 提升推拉 / D06 折叠 |
| 卧室普通窗（W 1200-2000mm） | W04 内开内倒 / W01+W04 一固一开 |
| 卧室飘窗（W ≥ 2400mm） | W01+W04+W01 三联（中固两侧开） |
| 厨房窗 | W04 内开内倒 / W05 上悬 |
| 卫生间窗 | W05 上悬 / W04 内开内倒 |
| 落地观景窗（高层） | W01 全固定 / W01+W04 大固定+小开启 |
| 楼梯间窗 | W01 全固定 |
| 入户门（单户） | D01 平开 / D02 双开 |

## 选择参考问题

如果客户犹豫不决，按这个顺序问：

1. 这个洞口主要是采光还是通风？
2. 高层吗？高层不建议外开。
3. 家里有小孩吗？有小孩优先内开内倒 + 限位器。
4. 隔音要求高吗？高的话避开推拉，优先平开/内倒。
5. 阳台要不要全开放？要的话考虑折叠门。
