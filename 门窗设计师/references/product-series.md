## 一、品牌框架（用于方案文档和客户对话）

- **品牌定位**：主动健康门窗
- **品牌表达**：下一代门窗，为下一代健康
- **核心心智**：能感知威胁、能思考策略、能主动执行净化/消杀/调节的"主动健康中枢"
- **品牌方向**：年轻人的第一樘门窗

### 产品哲学（4 维度）

| 维度 | 一句话 | 关键技术词 |
|------|--------|-----------|
| 健康 | 主动净化，如居森林 | ABG 光净化、元气森氧舱 Pro |
| 安全 | 智能预警，主动守护 | AI 云窗、风雨自关、抗菌安全执手 |
| 静音 | 动态降噪，智慧静谧 | 三道密封 + AI 通风调节 |
| 节能 | 光能驱动，智慧呼吸 | 光能充电、AI 联动空调 |
| 美学 | 智慧光影，定义场景 | 极窄边框、科技蓝/未来灰、智慧灯光 |

### 五大核心科技（在客户对话中可引用）

1. **ABG 光净化技术** — 阳光激活光触媒涂层，分解甲醛/苯/细菌/病毒
2. **AI 云窗智能管家** — 风雨自关、PM2.5/燃气监测、联动空调新风
3. **元气森氧舱 Pro** — 持续释放负氧离子（1000–3000 个/cm³）
4. **儿童抗菌安全系统** — 银离子抗菌执手 + 儿童安全锁
5. **光能充电 + 超强五金** — 日光供电免布线 + 配套五金承重 ≥120Kg

## 三、已录入系列详情

### PS5 ABG·AI 智慧系列

#### AGR112S — 极窄窗纱一体外开窗（AI 智慧）

**基础信息**

| 项 | 参数 |
|----|------|
| 占墙宽度 | 112mm |
| 属性归类 | 窗纱一体极窄边框外开窗 |
| 型材结构 | 6063-T5 高精级原生铝，壁厚 ≥1.8mm |
| 隔热条 | 上海优泰 PA66GF25-24mm |
| 隔音性能 | 35–43 dB |
| 玻璃配置 | 南玻 / 信义 / 福莱特（按批次） |
| 五金 | AMBERG 自研执手 + AMBERG 外开摩擦铰链 |
| 密封系统 | 江阴海达航空级三元乙丙胶条，三道密封，垂直等温线，销钉组角真注胶 |
| 安全防护 | 标配攀岩级安全绳，承重 100Kg |
| 设计细节 | 框扇齐平、隐藏式排水、极窄边框 |

**性能指标（六大维度）**

| 维度 | 指标 |
|------|------|
| 气密性 | 国标 8 级（最高） |
| 水密性 | 隐藏式排水 + 真注胶，28 天暴雨不渗漏 |
| 抗风压 | 抗 12 级台风 |
| 隔热保温 | 24mm 品牌隔热条，可省 20–30% 空调电费 |
| 隔音 | 多腔体 + 中空玻璃，38–42 dB |
| 五金承重 | ≥120Kg |

**官方色卡（外观可选色）**

- **标准色**：星河灰 / 月光白 / 青郦棕
- **定制色**：沉香褐 / 晨光金 / 镜湖蓝 / 远山灰 / 澜夜黑 / 暮云灰 / 云杉灰

**典型适用场景**

- 新装修家庭（甲醛分解需求）
- 三口之家、有老人/儿童的家庭（健康呼吸 + 抗菌）
- 沿海/高楼层（抗 12 级台风）
- 临街/临高架住宅（隔音需求）
- 颜值控、视野党（极窄边框 + 临江/临湖/园景房）
- 智能家居用户（远程控窗、与小米/华为/天猫精灵联动）
- 出差频繁的商务人士

**配套表面工艺**

- DX6000 新型复合膜涂层（耐腐蚀、耐氧化、耐高低温、不褪色）

## 六、使用规则（给门窗设计师 Agent 看）

1. **优先用已录入的系列**：如果客户的需求匹配 AGR112S（极窄/智能/外开/高隔音），可以直接推荐，并告诉客户"占墙 112mm、抗 12 级台风、隔音 38–42dB"等具体参数。
2. **未录入的系列不要捏造数据**：8 个系列中只有 PS5 已录入 1 款。其余 7 个系列在用户上传详细资料前，**不得编造参数**。如果客户问到，老实说"这个系列详细参数还在补充，您可以告诉我大致需求，我先记下"。
3. **品牌话术统一从本文件取**：方案文档和客户对话中的"主动健康门窗""下一代门窗，为下一代健康"这类品牌定位语，统一从"一、品牌框架"段引用，不要自己创造同义版本。
4. **机密资料处理**：PDF 文件本身标注"公司机密资料，未经书面许可不得擅自传阅、引用或复制"。本文件提取的是结构化技术数据用于内部 Agent 使用，不要把这些数据直接复制到对外发布的营销内容里。
